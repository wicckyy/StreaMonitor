## 2026-04-22 - [CRITICAL] Fix Insecure Direct Object Reference (IDOR) on Video Endpoint
**Vulnerability:** The `/video/<user>/<site>/<path:filename>` endpoint in `streamonitor/managers/httpmanager/httpmanager.py` exposed recorded video files directly without requiring authentication, allowing anyone with the URL structure to access sensitive media files. This is an Insecure Direct Object Reference (IDOR) vulnerability.
**Learning:** Even if surrounding endpoints (like the UI for listing videos) are protected by authentication, direct file serving endpoints must also explicitly include authentication checks. The Flask `@login_required` decorator must be applied to all routes handling sensitive data.
**Prevention:** Always apply the `@login_required` decorator to newly created routes that serve user-specific or sensitive data. Regular code review should ensure no route is left intentionally or accidentally public unless it's explicitly designed to be (e.g., static assets).
